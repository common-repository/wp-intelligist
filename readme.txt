WP Intelligist is a wordpress plugin that lets you embed your source codes from github, directly into blog posts or pages. It is built using Jquery-Intelligist and Chosen plugins available on Github.  

== Description ==

<strong>About WP Intelligist ?</strong>


You can use this plugin in both posts and pages using a simple shortcode. This was built to ease the code embedding functionality and keep it in an innovative way. I have used Jquery-Intelligist and Chosen dropdown search inside this plugin.


<strong>More Information and Support</strong>


http://www.innovativephp.com/wp-intelligist




== Installation ==

1. Upload `wp-intelligist` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to post or page editor and click on Gist Button in Visual Edior
4. Enter the number at end of gist codes and give relative title and click Add Code button
5. Once you add all the gist codes click Add Codes To Editor and publish the post.
6. You will see the embeded gist codes in the post or page.
